# Registration Template
The following project is a generic registration template.

## Quick Start
```bash
#Install dependencies
$npm install

#Start local server
$npm start

#Visit http://localhost:3000/
```

#### Functions:
1. Sign Up
2. Login
3. Deactivation

#### Dependencies:
- ExpressJS
- Handlebars
- Express-session
- PassportJS
- Sequelize
- Sqlite3
